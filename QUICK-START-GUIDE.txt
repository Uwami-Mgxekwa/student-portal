‚ö° QUICK SETUP (5 MINUTES):
================================================================================

1. CREATE STORAGE BUCKET IN SUPABASE:
   - Go to Supabase Dashboard > Storage
   - Click "Create bucket"
   - Name: resources
   - Public: YES ‚úì
   - Click "Create"

2. RUN SQL SETUP:
   - Go to Supabase Dashboard > SQL Editor
   - Open file: supabase-storage-setup.sql
   - Copy all SQL code
   - Paste and click "Run"
   - Wait for "Success"

3. TEST IT:
   - Start server: python -m http.server 8000
   - Go to: http://localhost:8000/admin/index.html
   - Login with your credentials
   - Click "File Manager"
   - Click "Upload File"
   - Upload a test PDF
   - Done! ‚úì

================================================================================
üìÅ HOW TO UPLOAD FILES:
================================================================================

1. Access admin panel: http://localhost:8000/admin/index.html
2. Click "File Manager" in sidebar
3. Click "Upload File" button
4. Fill in:
   - File Name: "Exam Timetable 2025"
   - Description: Optional
   - Category: Select (Timetables, Forms, Guides, etc.)
   - Year: Select (1, 2, 3, or All)
   - Choose PDF file (max 10MB)
5. Click "Upload File"
6. File is now live! Students can access it immediately.
